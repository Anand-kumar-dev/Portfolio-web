Raun

It features robust, token-based authentication (JWTs, refresh tokens) and a middleware that ensures seamless daily re-authentication to maintain continuous access to critical financial data like holdings, positions, and margins.

Jam

A full-stack, secure RAG Chatbot powered by Node.js/Express and MongoDB. It features an advanced Hybrid Search pipeline combining Qdrant (Vector DB) for semantic search and Elasticsearch for keyword search, with results intelligently reranked. It supports multiple Large Language Models (Gemini and Hugging Face) and uses Redis for robust, cookie-based JWT authentication and token blacklisting.
